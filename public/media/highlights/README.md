# Alaska Highlights Photos

Upload your Alaska trip photos here!

## File Naming

Name your photos as:
- `alaska-1.jpg`
- `alaska-2.jpg`
- `alaska-3.jpg`
- `alaska-4.jpg`
- `alaska-5.jpg`
- `alaska-6.jpg`
- (and so on...)

## Photo Guidelines

- **Format**: JPG or PNG
- **Recommended size**: 1200x900px or larger (landscape orientation works best)
- **File size**: Keep under 1MB each for fast loading
- **Content**: Your best photos from your Alaska internship trip

## Features

- Grid layout with hover effects
- Click to view full-size in lightbox
- Smooth animations
- Responsive design

The component will automatically display all photos you upload with the naming pattern above!
