# General Photos

Upload your general photos here!

## File Naming

Name your photos as:
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`
- `photo-5.jpg`
- `photo-6.jpg`
- (and so on...)

## Photo Guidelines

- **Format**: JPG or PNG
- **Recommended size**: 1200x900px or larger (landscape orientation works best)
- **File size**: Keep under 1MB each for fast loading
- **Content**: Your personal photos, memories, and moments

## Features

- Grid layout with hover effects
- Click to view full-size in lightbox
- Smooth animations
- Responsive design

The component will automatically display all photos you upload with the naming pattern above!
