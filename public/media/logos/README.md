# Company/Organization Logos

Upload logos for each organization in your experience section here.

## Required Logos

Place logo images in this folder with these exact filenames:

- `cmu.png` - Carnegie Mellon University logo
- `cmu-swim.png` - CMU Varsity Swim Team logo
- `cmu-hcii.png` - Carnegie Mellon HCII logo
- `uaa.png` - University of Alaska Anchorage logo
- `first-citizens.png` - First Citizens Bank logo

## Logo Guidelines

- **Format**: PNG with transparent background (recommended) or JPG
- **Size**: 200x200px to 400x400px (square aspect ratio works best)
- **Style**: White/light logos work best on dark backgrounds
- **File Size**: Keep under 100KB each for fast loading

## Fallback

If a logo is missing, the component will automatically fall back to the default icon for that experience entry.
